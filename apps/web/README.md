# apps/web

Next.js 15 (App Router, React 19). Phase 1 ships a single hello page; Phase 8
elaborates the full screen tree (Today, Insights, Library, Mock Exam, Onboarding,
Settings) per `08_FRONTEND_UX.md`.

## Run

```bash
pnpm --filter web dev      # http://localhost:3000
```

## Test

```bash
pnpm --filter web test     # vitest smoke
```
